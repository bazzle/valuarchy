# Valuarchy website v1

Static site built with Eleventy and Sass. Source is in `src/` and output is written to `_site/`.

## Prerequisites

- Node.js
- Yarn

## Install

```sh
yarn
```

## Development

```sh
yarn start
```

Runs Eleventy in serve mode and watches Sass, compiling `src/assets/styles/main.scss` to `_site/assets/styles/main.css`.

## Build

```sh
yarn build
```
